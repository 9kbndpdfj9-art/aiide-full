# DevView IDE（开发视界）

## 🌌 项目概述

**DevView IDE** 是全球首款**纯 Android 原生**的 AI 驱动集成开发环境。它不是 VS Code 的移动端移植，而是从零开始为触屏设备重新定义的开发范式。

**核心理念**：从"帮你写代码"转变为"帮你表达意图，并保证输出的正确性"。

**平台定位**：纯 Android 原生应用（无鼠标、无键盘依赖、全触屏优化、全自动化优先）

---
